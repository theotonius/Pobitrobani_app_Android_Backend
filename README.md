## ✨ Features Overview

পবিত্র বানী (Sacred Word) is an AI-powered spiritual companion for biblical wisdom. For a complete list of all features, please see the **[Features Documentation](features.html)**.

### Key Features

- **AI-Powered Search** - Search Bible verses, spiritual topics, hymns, and lyrics
- **3 Bible Versions** - Modern Bengali, Carey Version, Kitabul Mukkadas
- **Deep Verse Analysis** - Theological meaning, historical context, practical application
- **Multi-Language Support** - Bengali, English, and more
- **Dark/Light Theme** - Smooth animated transitions
- **Save & Organize** - Bookmark verses and insights
- **Responsive Design** - Works on all devices
- **Multi-Fallback AI** - Reliable service with Gemini, DeepSeek, Claude

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

---

## 🛠 Tech Stack

- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express
- **AI Providers**: OpenRouter (Gemini, Claude, DeepSeek)
- **Animations**: Framer Motion
- **Deployment**: Vercel
